Presentation Hub

Tus presentaciones,
en la web

Un sistema para crear, publicar y compartir presentaciones web con un solo comando

¿Qué es Presentation Hub?

Es una plataforma que convierte tus ideas en presentaciones web interactivas. Pedile a Claude que genere un deck sobre cualquier tema y en segundos tenés una URL pública lista para compartir.

Características

⚡

Generación instantánea

Claude crea presentaciones completas con diseño profesional en segundos

🎨

Diseños únicos

Cada presentación tiene un estilo visual diferente — nunca se repiten

🔗

URLs compartibles

Cada deck tiene su URL pública que podés enviar a cualquier persona

📱

Responsive

Se ven perfectas en desktop, tablet y mobile

⌨️

Navegación intuitiva

Flechas del teclado, click, o indicadores para moverse entre slides

🚀

Deploy automático

Se publica automáticamente vía GitHub y Railway

¿Cómo funciona?

1

Pedí tu deck

Decile a Claude sobre qué querés la presentación

2

Se genera el HTML

Claude crea un archivo autocontenido con diseño único

3

Se sube a GitHub

El archivo se pushea al repositorio automáticamente

4

Railway deploya

En 1-2 minutos tu presentación está live con URL pública
